Projeto desafio para a ProBrain

#Números

Você deverá criar uma aplicação simples, com 4 diferentes implementação do “Números” 
utilizando HTML, Sass e JavaScript, e a ferramenta Jsfiddle. 
O usuário deverá ter a opção de escolher entre as 4 implementações (versões do jogo) 
inserir um número de 1 - 1000. Sua aplicação, então, deverá exibir o resultado de todas as 
implementações para o usuário. 

Para essa tarefa você deverá utilizar pelo menos dois dos princípios SOLID.

Implementação 1 - Se o número é par ou ímpar.
Implementação 2 - Se o número é primo ou não é primo.
Implementação 3 - Mostrar se o número existe na sequência de Fibonacci.
Implementação 4 - Mostrar a sequência de números onde múltiplos de 4 utilizem a palavra “pin”.
